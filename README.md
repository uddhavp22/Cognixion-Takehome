# Converse

A hover-driven recipe explorer built in Unity 6. Designed as a take-home for Cognixion — every interaction works without a click, using dwell-to-commit as a stand-in for BCI/gaze input.

## Quick start

1. Clone the repo
2. Copy `UnityProject/Assets/StreamingAssets/appsettings.example.json` to `appsettings.local.json` in the same folder and add your Anthropic API key
3. Open `UnityProject/` in Unity 6 (LTS)
4. Press Play from the `01_Constraints` scene

> **No API key?** The app falls back to 10 built-in recipes automatically.
